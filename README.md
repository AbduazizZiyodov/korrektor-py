## **Installation**

Using `pip`:

```bash
pip3 install --upgrade korrektor-py
```

Build & install from source code:

```
git clone https://github.com/AbduazizZiyodov/korrektor-py.git
cd korrektor-py/

python3 setup.py sdist bdist_wheel && cd dist/
pip install Korrektor_Py-*.whl

```
